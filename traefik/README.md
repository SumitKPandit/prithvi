# Traefik

Traefik acts as the reverse proxy on home network. It directs traffic coming for specific services, identified as <service_name>.lan.sumitkpandit.in to respective service correct port.

Traefik dashboard can be accessed on [traefik.lan.sumitkpandit.in](traefik.lan.sumitkpandit.in)

## Installation Steps

To install traefik follow below steps:

1. SSH into the private server
2. Make sure Docker and Docker Compose is installed
3. Make sure a docker network named `traefik-network` is available, if not create one with following command
```bash
docker network create traefik-network
```
4. Clone this repository using following command
```bash
git clone git@github.com:SumitKPandit/prithvi.git
```
5. Change directory into where traefik's compose file is located.
```bash
cd ./prithvi/traefik/
```
6. We persistently store Let's Encrypt certificates generated by Traefik so we loose them everytime service is restarted. For this to work properly, set appropriate permissions for the `cloudflare-acme.json` file.
```bash
chmod 600 ./data/certs/cloudflare-acme.json
```
7. Create a `.env` file beside `compose.yml` file and set `CF_DNS_API_TOKEN`. This is required for completing DNS Challenge for issuing SSL certificates.
8.  Start the service
```bash
docker compose up -d
```
